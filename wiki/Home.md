# Welcome to the NyxOS Wiki

**NyxOS** is a from-scratch x86_64 operating system written in C and Assembly. It features preemptive multitasking, a windowed GUI compositor, a full TCP/IP networking stack, EXT2 filesystem support, SMP, demand paging, copy-on-write, and userspace (ring 3) ELF loading.
